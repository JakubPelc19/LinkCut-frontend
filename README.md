# LinkCut-Frontend
A modern frontend for LinkCut, a URL shortener built with Next.js, TypeScript, and a clean, responsive UI.
This frontend communicates with the LinkCut ASP.NET Core API to shorten links and display results.

## Tech Stack
- Next.js

- React

- TypeScript

- Tailwind CSS

- Fetch API for backend communication

## Features
- Submit a URL and receive a shortened version

- Simple, fast, and responsive UI

- Client-side validation

- Smooth UX

## Setup
1. Clone the repository: `git clone https://github.com/JakubPelc19/LinkCut-frontend`

2. Install dependencies:
```bash
npm install
# or
pnpm install
# or
yarn install
```
3. Configure env variable (create `.env` file):
```
LINKCUT_API=https://<url-to-linkcut-api>
```
4. You can start it now :)
